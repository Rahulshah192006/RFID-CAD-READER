## RFID CARD READER
This Can read the card and print the id

## Things needed 
1: Audrino UNO 
2:RC522 RFID 13.56MHZ Reader Writer Module
3:Audrino IDE

## Ports and Pins
---RFID MODULE  Audrino pin
    3.3v         3.3v
    RST          AudrinoUno 2
    GND          Audrino GND
    MISO         AudrinoUno 12
    MOSI         AudrinoUno 11
    SCK          AudrinoUno 13
    SDA          AudrinoUno 10
